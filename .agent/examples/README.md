# Protocol Examples

This directory stores non-runtime examples for task, context, run, plan, and
governance records.

Files here are reference material only. New project workspaces should not treat
them as active task state, active directives, pending escalations, run history,
or assembly inputs.

Runtime project records belong in the corresponding top-level directories:

- `tasks/`
- `context/`
- `runs/`
- `governance/`
- `plan/`
